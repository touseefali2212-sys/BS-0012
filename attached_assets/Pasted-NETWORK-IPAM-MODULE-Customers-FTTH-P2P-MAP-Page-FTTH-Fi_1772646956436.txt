1️⃣ FTTH Fiber Route Map (Editable)

Provides full fiber infrastructure planning and editing tools.

Features:

✔ Draw Fiber Route (Polyline Tool)

✔ Define Fiber Core Count
✔ Auto Distance Calculation
✔ Route Labeling (Route ID)
✔ Assign Route to OLT
✔ Attach Route to Splitter

Route Information Panel:

Route Name

Total Length (Meters/KM)

Fiber Capacity

Used Fibers

Available Fibers

Connected Customers

Advanced Tools:

✔ Split Fiber Route
✔ Extend Route
✔ Merge Routes
✔ Mark Joint Closure
✔ Mark Distribution Box
✔ Route Utilization Indicator

This allows real-time fiber expansion planning.
