ACCOUNTING MODULE
Budget Allocation Page

Project: NetSphere ISP Billing & Enterprise ERP

🎯 Objective

Build a professional “Budget Allocation” page inside the Accounting module.

This page must allow Admin/Finance to:

Create annual / monthly budgets

Allocate budget to departments

Assign budgets to expense accounts

Allocate branch-wise budgets

Track Budget vs Actual

Control overspending with alerts

This page turns your system into a Financial Planning & Control ERP.

🔄 System Logic

When Expense Entry is posted:

✔ System checks budget allocation
✔ Deducts from allocated budget
✔ Updates remaining balance
✔ Changes status indicator
✔ Generates alert if threshold reached

When Budget Period Ends:

✔ Lock new allocations
✔ Archive budget
✔ Generate budget performance report
