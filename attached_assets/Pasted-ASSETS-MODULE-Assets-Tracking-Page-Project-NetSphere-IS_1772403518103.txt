ASSETS MODULE
Assets Tracking Page

Project: NetSphere ISP Billing & Enterprise ERP

🎯 Objective

Develop a real-time Assets Tracking Page that provides complete visibility into the location, status, ownership, condition, and lifecycle of all organizational assets.

This page ensures:

✔ Real-time asset location tracking
✔ Chain-of-custody transparency
✔ Customer-linked device visibility
✔ Warehouse & POP inventory clarity
✔ Technician accountability
✔ Loss prevention & asset recovery

This page acts as the operational command center for infrastructure monitoring.

🔄 Automation Logic

When asset transferred:

✔ Location auto-updated
✔ Status updated
✔ Movement logged

When assigned to customer:

✔ Linked to service
✔ IP allocated
✔ Status updated

When marked faulty:

✔ Move to maintenance queue
✔ Flag in dashboard

When returned:

✔ Clear assignment
✔ Update location
✔ Adjust condition

If lost:

✔ Lock status
✔ Alert admin
✔ Log investigation
