NOTIFICATIONS MODULE
Push SMS Page

(SIM SMS | Email SMS | WhatsApp Messaging)

Project: NetSphere ISP Billing & Enterprise ERP

🎯 Objective

Develop a centralized Push Messaging Console to send real-time, direct notifications to customers or internal users using:

✔ SIM-Based SMS (Device GSM)
✔ Email-to-SMS Gateway
✔ WhatsApp Messaging API

This page enables:

✔ Instant manual message dispatch
✔ Multi-channel broadcasting
✔ Customer-group targeting
✔ Delivery tracking
✔ Emergency communication

It acts as the rapid response communication engine of the ERP.

📊 Page Structure

The page contains 5 structured sections:

1️⃣ Channel Selection & Status
2️⃣ Recipient Targeting
3️⃣ Message Composer
4️⃣ Send & Scheduling Controls
5️⃣ Delivery Monitoring & Logs

1️⃣ Channel Selection & Status

Select communication channel:

✔ SIM-Based SMS

Send SMS using GSM modem or SIM device connected to system.

Fields:

SIM Device Name

Connected Port

Signal Strength

SIM Balance Status

Daily Limit

Features:

✔ Auto SIM Detection
✔ Multi-SIM Rotation
✔ Failover SIM

✔ Email-to-SMS Gateway

Send SMS through email-based telecom gateway.

Fields:

SMTP Server

Email Account

Gateway Domain

Sender Name

Supports international format.

✔ WhatsApp Messaging

Official API-based messaging via WhatsApp Business Platform

Fields:

Template Selection

Media Attachment

CTA Button

Auto Personalization Variables

Supports approved templates only.

🔄 Automation Flow

When message sent:

System validates recipients

Applies personalization variables

Checks channel availability

Sends message

Logs status

Applies fallback if failed

All activities recorded in Activity Log.
