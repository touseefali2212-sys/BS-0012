🔄 Automation Logic

When allocation approved:

✔ Reduce available stock
✔ Mark assets as “Reserved”
✔ Notify destination

If converted to transfer:

✔ Auto-create Transfer Request

If converted to assignment:

✔ Open Assignment Workflow

If reservation expires:

✔ Return stock to available pool
✔ Notify requester

If stock below threshold:

✔ Trigger low-stock alert
