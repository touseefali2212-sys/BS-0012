🔄 System Logic

When project created:

✔ Assign unique project ID
✔ Link to department
✔ Enable task creation under project
✔ Allocate budget

When expense recorded:

✔ Deduct from project budget
✔ Update accounting
✔ Update dashboard KPIs

When all tasks completed:

✔ Auto-mark project Completed
✔ Generate completion report
