Reseller Management
Wallet & Transactions Page
Project: NetSphere ISP Billing & Enterprise ERP
________________________________________
🎯 Objective
Build a professional Wallet & Transactions Management page for resellers.
The system must allow:
•	View reseller wallet balance
•	Recharge wallet
•	Deduct balance automatically
•	Track all transactions
•	Apply commission credits
•	Monitor credit limits
•	Generate financial reports
•	Export statements
•	Integrate with accounting
This page must integrate with:
✔ Reseller List
✔ Package Sales
✔ Commission System
✔ Accounting Module
✔ Vendor Billing

💰 Wallet Logic (Important)
Support two models:
________________________________________
1️⃣ Prepaid Model
•	Reseller must recharge first
•	Wallet deducted when:
o	Customer activated
o	Customer renewed
o	Package upgraded
•	If balance insufficient:
→ Block activation
________________________________________
2️⃣ Postpaid Model
•	Allow negative balance up to credit limit
•	Monthly invoice generated
•	Auto alert when near credit limit
•	Auto suspend if limit exceeded
