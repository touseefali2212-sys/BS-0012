ASSETS MODULE
Assets List Page

Project: NetSphere ISP Billing & Enterprise ERP

🎯 Objective

Develop a centralized Assets List Page to manage, monitor, and track all physical and logical assets across the organization from procurement to retirement.

This page ensures:

✔ Real-time inventory visibility
✔ Deployment tracking
✔ Serial & MAC tracking
✔ Location monitoring
✔ Asset lifecycle management
✔ Financial depreciation tracking
✔ Maintenance scheduling control

This is your complete infrastructure asset monitoring system.

🔄 Automation Logic

When new asset added:

✔ Assign unique Asset ID
✔ Apply asset type rules
✔ Link depreciation method

When assigned to POP:

✔ Update location
✔ Link to network device

When assigned to customer:

✔ Link with customer profile
✔ Mark deployed

When warranty < 30 days:

✔ Send alert
✔ Notify admin

When retired:

✔ Stop depreciation
✔ Update financial records
