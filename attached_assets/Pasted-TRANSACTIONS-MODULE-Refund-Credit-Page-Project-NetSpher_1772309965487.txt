🔄 System Logic

When Refund is Approved:

✔ Reduce revenue
✔ Adjust invoice balance
✔ Update customer wallet (if selected)
✔ Create accounting journal entry
✔ Update financial reports
✔ Update dashboard KPIs

If converted to credit:

✔ Create Credit Note
✔ Apply to future invoice
✔ Update Accounts Receivable

If refunded to bank:

✔ Reduce bank balance
✔ Record payment outflow
