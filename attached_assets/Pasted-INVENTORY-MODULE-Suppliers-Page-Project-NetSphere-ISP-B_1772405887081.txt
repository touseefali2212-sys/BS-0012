🔄 Automation Logic

When Purchase Order created:

✔ Linked to selected supplier
✔ Updates supplier purchase history

When invoice approved:

✔ Updates outstanding payable

When payment recorded:

✔ Reduces supplier balance

If credit limit exceeded:

✔ Block new PO creation
✔ Notify finance

If supplier marked inactive:

✔ Prevent new transactions

🔐 Role-Based Access

Admin:
Full supplier control

Procurement Manager:
Add/edit suppliers

Finance Manager:
Manage credit & payments

Inventory Manager:
Create purchase orders

Viewer:
Read-only

Only Admin can permanently blacklist supplier.
