HRM – Staff User Login Page

Project: NetSphere ISP Billing & Enterprise ERP

🎯 Objective

Design and build a professional, secure, modern Staff User Login page for HRM system.

The login system must:

Support Office & Field staff

Be role-based

Be secure (OTP / device tracking)

Support multi-branch login

Support mobile & desktop responsive UI

Log login history

Restrict access based on Role & Area

This page must feel premium and enterprise-level.

🔄 System Login Logic

When user logs in:

System must:

1️⃣ Validate credentials
2️⃣ Check Role
3️⃣ Check App Access Permission
4️⃣ Check Area Assignment (if field user)
5️⃣ Load permitted modules only
6️⃣ Apply branch filtering
7️⃣ Record login session

If Recovery Officer:
→ Redirect to Recovery Dashboard

If HR:
→ Redirect to HR Dashboard

If Admin:
→ Redirect to Admin Panel

If Technician:
→ Redirect to Ticket Dashboard
