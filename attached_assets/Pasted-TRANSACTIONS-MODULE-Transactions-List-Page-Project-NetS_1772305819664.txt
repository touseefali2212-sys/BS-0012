🔄 System Logic

When a transaction is created from any module:

✔ It must appear here automatically
✔ Debit & Credit must balance
✔ Must show module source
✔ Must update dashboard instantly
✔ Must link to journal entry
✔ Must respect period lock rules

If reversed:

✔ Create reversing journal entry
✔ Maintain audit history
✔ Change status to Reversed
