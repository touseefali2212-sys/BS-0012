🔄 System Logic
When customer pays online:
✔ Gateway sends callback
✔ System verifies transaction
✔ Marks invoice as paid
✔ Creates income journal entry
✔ Debits Bank Account
✔ Credits Accounts Receivable
✔ Records gateway charges as expense
✔ Updates dashboard
If refund processed:
✔ Reverse journal entry
✔ Update invoice status
✔ Adjust AR
________________________________________
📊 ISP Use Case Example
Customer pays 15,000 PKR via online gateway.
Gateway charges 2% (300 PKR).
System auto posts:
Debit: Bank 14,700
Debit: Gateway Charges Expense 300
Credit: Accounts Receivable 15,000
Fully automated.
