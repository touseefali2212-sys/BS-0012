This page ensures:

This is the backbone of scalable ISP network architecture.

🔄 Automation Logic

When new customer created:

✔ Assign IP from selected pool
✔ Link VLAN automatically
✔ Sync with MikroTik / BRAS

When IP pool utilization > 80%:

✔ Generate alert
✔ Notify Network Admin

When subnet exhausted:

✔ Mark as Full
✔ Prevent further allocation

When VLAN disabled:

✔ Block new assignments
✔ Notify network engineer
