NETWORK & IPAM MODULE
Customer MAP Page

Project: NetSphere ISP Billing & Enterprise ERP

🎯 Objective

Develop an interactive Customer Map Page that visualizes all subscribers geographically, linked with POPs, routers, VLANs, and IP pools.

This page enables:

✔ Real-time customer location tracking
✔ POP coverage visualization
✔ Network density analysis
✔ Fault impact identification
✔ Area-wise revenue insights
✔ Expansion planning
✔ Field team navigation support

This is your network footprint intelligence dashboard.

📊 Page Layout Structure

The page contains 6 main components:

1️⃣ Map Overview Panel
2️⃣ Customer Geo Markers
3️⃣ POP & Infrastructure Layer
4️⃣ Heatmap & Density View
5️⃣ Area Analytics Panel
6️⃣ Filter & Drill-Down Controls

1️⃣ Map Overview Panel

Top Summary Cards:

Total Customers

Active Customers

Suspended Customers

Corporate Clients

POP Count

Coverage Areas

Quick Stats:

📊 Customers per City
📈 Growth by Area
📊 Revenue by Region

2️⃣ Customer Geo Markers

Each customer appears as a location marker.

Clicking a marker shows:

Customer Name

Account ID

Package Speed

Current Status

IP Address

VLAN

POP / Router

Signal Type (FTTH / Wireless)

Installation Date

Outstanding Balance

Last Activity

Quick Actions:

✔ View Profile
✔ Suspend / Activate
✔ Create Task
✔ Navigate via Google Maps

3️⃣ POP & Infrastructure Layer

Show network infrastructure on map:

✔ POP Locations
✔ OLT Locations
✔ MikroTik Routers
✔ Distribution Boxes
✔ Fiber Routes (Optional Layer)
✔ Wireless Towers

Each POP shows:

Total Connected Users

Capacity

Utilization %

Congestion Status

Color Coding:

🟢 < 60% utilization
🟡 60–80%
🔴 > 80%

🔄 Automation Logic

When new customer added:

✔ Auto-pin location
✔ Assign POP connection
✔ Sync with IPAM

When customer suspended:

✔ Marker color changes
✔ Update in real-time

When POP overloaded:

✔ Highlight POP in red
✔ Notify admin
✔ Suggest expansion zone

When complaint created:

✔ Highlight customer marker
✔ Show issue icon
