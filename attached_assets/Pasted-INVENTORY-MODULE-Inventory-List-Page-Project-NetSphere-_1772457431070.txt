This page ensures:

✔ Complete stock visibility
✔ Real-time quantity tracking

✔ Stock health monitoring

✔ Reorder control

It becomes the core operational stock command center.

🔄 Automation & System Logic

When Purchase Order is received:

✔ Stock increases
✔ Cost updated (based on costing method)
✔ Inventory valuation recalculated

When Sale Invoice is created:

✔ Stock decreases
✔ COGS recorded

When Transfer occurs:

✔ Decrease from source warehouse
✔ Increase in destination warehouse

When Allocation happens:

✔ Move quantity from Available → Allocated

If stock falls below reorder level:

✔ Auto-generate reorder alert
✔ Notify inventory manager

Supports:

✔ FIFO
✔ Weighted Average Cost
✔ Serial Tracking
✔ Batch Tracking
