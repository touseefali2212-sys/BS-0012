TASK MANAGEMENT MODULE
Task Management Page

Project: NetSphere ISP Billing & Enterprise ERP

🎯 Objective

Develop a centralized Task Management Page that provides complete visibility and control over all operational, technical, field, and administrative tasks across the organization.

This page enables:

✔ Real-time monitoring of tasks
✔ Team accountability
✔ Deadline enforcement
✔ Performance evaluation
✔ Project execution tracking
✔ Operational efficiency

This is the main execution control panel of the company.

🔄 System Logic

When Task Created:

✔ Unique Task ID auto-generated
✔ Linked to Project (if selected)
✔ Assigned user notified
✔ Appears in dashboard

When Status Updated:

✔ Activity log recorded
✔ Supervisor notified
✔ Progress recalculated

When Overdue:

✔ Automatically marked overdue
✔ Alert notification sent
✔ Highlighted in red

When Completed:

✔ Completion time recorded
✔ Project progress updated
✔ Employee KPI updated

🔐 Role-Based Access Control

Admin:
Full Control

Project Manager:
Create / Assign / Monitor

Supervisor:
View team tasks

Employee:
Update assigned tasks only

Viewer:
Read-only access

Task deletion only by Admin.
