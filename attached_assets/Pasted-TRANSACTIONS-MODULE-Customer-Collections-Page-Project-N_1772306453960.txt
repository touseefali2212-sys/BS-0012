🔄 System Logic

When collection is recorded:

✔ Reduce customer outstanding
✔ Update AR aging
✔ Create journal entry
✔ Update dashboard
✔ Update collection performance
✔ Close invoice if fully paid
✔ Reflect in Trial Balance

If partial:

✔ Update invoice balance
✔ Change status to Partial

If overdue and paid:

✔ Remove suspension flag
